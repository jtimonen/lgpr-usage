Out-of-sample prediction using the lgpr package¶

This notebook demonstrates computing, visualizing and evaluating model predictions at separate test points. First we create some data and split it into training and test points.

In [1]:
require(lgpr)
require(repr)
require(ggplot2)

set.seed(123)
simData <- simulate_data(N            = 16,
                         t_data       = seq(6, 36, by = 4),
                         covariates   = c(    2,2),
                         lengthscales = c(6,6,6,6),
                         relevances   = c(1,1,1,0),
                         names        = c("sex", "location"),
                         t_jitter     = 0.2,
                         snr          = 5)

data  <- simData$data
split <- lgpr:::split_data_random(data, p_test = 0.2)
plot_simdata(simData, i_test = split$i_test)
