This vignette demonstrates how to use the lgp function of the lgpr package as an alternative for the the linear mixed effect model fitting function lmer of the lme4 package Bates et al. (2015).

Data simulation

We use lgpr to simulate data with nonlinear effects.

set.seed(1323)
simData <- simulate_data(N            = 16,
                         t_data       = c(12,24,36,48,60),

Linear mixed effect modeling using lme4

We use the lmerTest package by Kuznetsova, Brockhoff, and Christensen (2017) to fit and study a linear mixed effects model. It uses the lmer function of the lme4 package by Bates et al. (2015) to fit the model, but extends lme4 with more comprehensive model selection and assessment methods.

fit <- lmerTest::lmer(y ~ 1 + id + (age|id) 
                      + z1 + z2 + z3,
                      data = data)
## boundary (singular) fit: see ?isSingular

Additive GP modeling using lgpr

Then, we fit an additive GP model using lgpr.

                  data    = data,
                  iter    = 1000, 
                  chains  = 1,
                  refresh = 1000)
