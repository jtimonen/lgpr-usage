1. Simulating data¶

In this example we generate count data with 10 individuals and 8 measurement time points for each.

2. Fitting the model¶

In [3]:
N_ITER <- 1000
fit <- lgp(formula   = y ~ id + age +  x1 + x2 + z1 + z2,
          data       = simData$data,
          likelihood = "NB",
          iter       = N_ITER, 
          parallel   = TRUE, 
          chains     = 4,
          control    = list(adapt_delta = 0.95),
          save_warmup = FALSE,
          refresh = 0,
          verbose = TRUE)
